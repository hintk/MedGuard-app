AI 智能识别与分析

拍照自动识别药盒信息 (Doubao-Seed-1.6-vision)；将所有在用药品发给豆包大模型，由 AI 临床药师全面评估药物相互作用风险，输出高/中/低风险等级及详细建议。支持用户自行更换更高端模型。

6 位 PIN + 面容 ID 安全体系

首次注册强制设置 6 位数字密码 (SHA-256 + 随机盐 + Keychain 硬件加密)。日常 Face ID / Touch ID 无感解锁，敏感设置需 PIN 二次验证。全部数据仅存本地，不上传云端。
